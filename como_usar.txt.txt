📦 O QUE FAZER PRIMEIRO:

1. ✅ INSTALAR PROGRAMAS NECESSÁRIOS:
   - Baixar Dart: https://dart.dev/get-dart
   - Instalar MySQL: https://dev.mysql.com/downloads/
   - VS Code: https://code.visualstudio.com/

2. 🗄️ CONFIGURAR BANCO DE DADOS:
   - Abrir o MySQL
   - Executar o arquivo "entrega8.sql"
   - Pronto! Banco criado automaticamente

3. 🔧 CONFIGURAR PROJETO DART:
   - Abrir pasta no VS Code
   - No terminal, digitar: dart pub get
   - Isso baixa tudo que precisa

4. 🚀 EXECUTAR:
   - No terminal digitar: dart run bin/server.dart
   - Abrir navegador em: http://localhost:8080

📊 O QUE O SISTEMA FAZ:
- Monitora sensores de temperatura e movimento
- Controla lâmpadas automaticamente
- Mostra dados em tempo real
- Grava tudo no banco de dados

🔌 URLs IMPORTANTES:
http://localhost:8080/leituras          → Ver dados
http://localhost:8080/estatisticas/1    → Ver filial 1
http://localhost:8080/estatisticas/2    → Ver filial 2

❌ PROBLEMAS COMUNS:
- Erro MySQL? Verificar se o MySQL está rodando
- Erro Dart? Executar "dart pub get" de novo
- Porta ocupada? Mudar porta no arquivo server.dart

💡 DICA: Sempre que abrir o projeto, executar "dart pub get"
